# User Activity Monitor

Программное средство для мониторинга пользовательской активности, классификации записей активности с помощью ML-модели, хранения истории, просмотра статистики и экспорта отчетов.

## Возможности

- запуск и остановка мониторинга активности;
- сбор данных об активном окне, процессе, действиях клавиатуры и мыши;
- определение домена активного сайта;
- классификация активности с помощью обученной модели;
- хранение записей активности в локальной базе данных;
- просмотр истории активности;
- формирование статистики;
- экспорт отчетов;
- настройка категорий и параметров мониторинга;
- управление файлом обученной ML-модели.

## Запуск из исходного кода

```bash
pip install -r requirements.txt
python -m app.ui.app
```

## Консольный запуск мониторинга

```bash
python -m app.main
```

## Переобучение модели

```bash
python -m app.tools.retrain_model --dataset training_dataset.csv --output-dir models/new_model
```

## Структура проекта

- `app/ui/` - графический интерфейс;
- `app/monitoring/` - сбор пользовательской активности;
- `app/ml/` - классификация активности;
- `app/repositories/` - работа с хранилищем данных;
- `app/services/` - сервисы статистики, отчетов и сохранения записей;
- `app/tools/` - вспомогательные инструменты;
- `features.py` - формирование признаков для ML-модели.

## Пользовательская сборка

Готовая сборка приложения может размещаться в разделе GitHub Releases в виде архива с исполняемым файлом. В этом случае пользователю достаточно скачать архив, распаковать его и запустить приложение.
